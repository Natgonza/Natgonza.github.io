Game Rules

  • Player 1 always starts on page reset.
  • Players take turns to drop colored discs into a 7x6 grid.
  • The first player to connect four discs is a row, either horizonally, vertically, or diagonally.
  • The first player to connect four discs wins the game.
  • If the grid is filled without a winner, the game is a draw.
  • Click the "Reset Game" button to start a new game.
  • The last player to win goes first the next round.
